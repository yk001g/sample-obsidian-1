# ルールファイル更新プロンプト

## 使用タイミング
システムルールやコマンドファイルを更新した時、AGENTS.mdにも反映する必要がある時

## プロンプト本文
@AGENTS.md にもそのルール書いといて

または、より詳細に：

この変更をAGENTS.mdにも反映してください。以下の内容を含めてください：
- [変更内容の説明]
- [影響を受けるフォルダ/ファイル]
- [新しいルールの詳細]

## 使用例
```
@AGENTS.md にもそのルール書いといて
```

または

```
この変更をAGENTS.mdにも反映してください。
新しいフォルダ構造: 08_prompts/ を追加
カテゴリ: 01_整理系、02_生成系、03_レビュー系、04_学習系、05_改善系、06_一括作業系
```

## 期待される結果
- AGENTS.mdに新しいルールが追加される
- 既存のルールと整合性が保たれる
- フォルダ構成の説明が更新される

## 注意事項
- 既存のルールとの整合性を確認
- フォルダ構成の説明も更新
- 具体例も追加すると良い
