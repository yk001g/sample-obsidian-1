# Memory Note作成支援プロンプト

## 使用タイミング
InboxノートをMemory Note（長期記憶）に昇華する時

## プロンプト本文
指定されたInboxノートをMemory Noteに昇華してください。

引数:
- Inboxファイル名: 変換元のファイル（ユーザーが指定）

実行内容:
1. 指定ファイルの内容を読み込む
2. 以下の基準で評価:
   - 自分の言葉で書かれているか
   - 1ノート1アイデアか
   - 完結しているか
   - 適切なリンクがあるか
   - タグが適切か
   - 3ヶ月後も参照する価値があるか
3. 改善案を提示
4. ユーザーの承認後に `04_Memory/` の適切なサブフォルダに移動

評価基準 (10点満点):
- 自分の言葉: 3点
- 1ノート1アイデア: 2点
- 完結性: 2点
- リンク: 2点
- タグ: 1点

手順:
1. 指定ファイルを読み込む
2. 評価を行う
3. 改善案を提示
4. ユーザーの承認を待つ
5. 承認後、`04_Memory/`の適切なサブフォルダに移動
   - AI/（AI関連）
   - Business/（ビジネス関連）
   - Education/（教育関連）
   - Personal/（個人関連）
   - Technical/（技術関連）
6. 該当するMOC（Map of Contents）を更新

## 使用例
```
/make-memory 20241022_API設計メモ.md
```

## 期待される結果
- ファイルが評価される（スコア表示）
- 改善案が提示される
- ユーザー承認後、適切なMemoryサブフォルダに移動される
- 該当するMOCが更新される

## 注意事項
- 3ヶ月後も参照する価値があるかを必ず確認
- 品質スコア7点以上が目安
- MOCの更新を忘れずに
