# npm run test解決プロンプト

## 使用タイミング
npm run testでエラーが出た場合に、すべてのテストが通るまで自動的にテストソースを修正したい時

## プロンプト本文
```
/Users/akira_funakoshi/work_space/apl
でnpm run testでエラーが出る、すべてテストが通るまでテストソースを修正して、その作業を繰り返して。

これnpm run testでビルド通るようにして。
そのコマンド打ってエラー出たらすべてテストが通るまでテストソースを修正して。
コマンドでエラー無くなるまで修正して。
永遠に治るまで作業して
```

## 使用例
プロジェクトのパスを指定して、npm run testのエラーを自動修正する場合

## 期待される結果
1. プロジェクトのパスが指定される
2. `npm run test`が実行される
3. エラーが出たら自動的にテストソースが修正される
4. すべてのテストが通るまで繰り返される

## 注意事項
- プロジェクトのパスは実際のパスに置き換える
- テスト自動修正のベストプラクティスを参照
- 無限ループに注意
