# Next.js化プロンプト

## 使用タイミング
既存のリポジトリをNext.jsのApp Router構成に変換したい時

## プロンプト本文
```
このリポジトリを網羅的に理解して。docs/tasksフォルダに
今回のnextjs化タスクのフォルダを作成して、
このリポジトリの内容を抽象化して整理したmdファイルを作成して、
その後に、nextjs化するための注意事項や手順を詳細に書いたmdファイル作成して

その後に、実際に、このリポジトリをnextjsのapp router構成に変換して
```

## 使用例
既存のReactアプリケーションをNext.js App Router構成に変換する場合

## 期待される結果
1. リポジトリが網羅的に理解される
2. `docs/tasks`フォルダにNext.js化タスクのフォルダが作成される
3. リポジトリの内容を抽象化して整理したmdファイルが作成される
4. Next.js化するための注意事項や手順を詳細に書いたmdファイルが作成される
5. 実際にリポジトリがNext.jsのApp Router構成に変換される

## 注意事項
- Next.js App Router移行のベストプラクティスを参照
- 既存の機能が壊れないように注意
- 段階的に移行することを推奨
