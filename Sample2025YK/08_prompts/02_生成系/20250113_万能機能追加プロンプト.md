# 万能機能追加プロンプト

## 使用タイミング
任意の機能を追加したい時

## プロンプト本文
```
以下のタスクを実行したい。

今日のフォルダのtasksの中にそのタスクの「タスク ID+タスクタイトル」の日本語名フォルダ作成して、その中にmdファイル作成してユーザーニーズと仕様要件と処理手順設計をわかりやすく書いて。
その後に、その通りに実装して。
＝＝＝＝＝
{追加したい機能の概要}
```

## 使用例
新機能「ユーザー認証機能」を追加する場合：
```
以下のタスクを実行したい。

今日のフォルダのtasksの中にそのタスクの「タスク ID+タスクタイトル」の日本語名フォルダ作成して、その中にmdファイル作成してユーザーニーズと仕様要件と処理手順設計をわかりやすく書いて。
その後に、その通りに実装して。
＝＝＝＝＝
ユーザー認証機能を追加したい。ログイン、ログアウト、パスワードリセット機能を含む。
```

## 期待される結果
1. `{追加したい機能の概要}`の部分に追加したい機能の説明が記入される
2. `docs/tasks`フォルダ内に「タスク ID+タスクタイトル」の日本語名フォルダが作成される
3. その中にmdファイルが作成され、ユーザーニーズと仕様要件と処理手順設計が書かれる
4. その通りに実装される

## 注意事項
- 機能追加のワークフローを標準化する
- ユーザーニーズと仕様要件を明確にする
- 処理手順設計を詳細に書く
