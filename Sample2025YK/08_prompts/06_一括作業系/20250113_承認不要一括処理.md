# 承認不要一括処理プロンプト

## 使用タイミング
ユーザーの承認が不要で、一気に処理を実行する時

## プロンプト本文
ユーザーの承認いらないから一気にやって

または、より具体的に：

以下の処理を承認不要で一気に実行してください：
- [処理1]
- [処理2]
- [処理3]
...

実行後、結果をまとめて報告してください。

## 使用例
```
ユーザーの承認いらないから一気にやって
```

または

```
以下の処理を承認不要で一気に実行してください：
- 00_Memo内の全ファイルを01_Inbox形式に変換
- 元のファイルを99_Archiveに移動
- 空のフォルダを削除

実行後、変換レポートを表示してください。
```

## 期待される結果
- 処理が一気に実行される
- ユーザーの承認を待たずに完了する
- 実行結果がまとめて報告される

## 注意事項
- 破壊的な操作の場合は確認が必要な場合もある
- 実行前に処理内容を明確に説明
- 実行後は結果を必ず報告
