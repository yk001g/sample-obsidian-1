# アーカイブレビュープロンプト

## 使用タイミング
四半期ごと（3ヶ月に1回）に3ヶ月以上前のアーカイブをレビューする時

## プロンプト本文
3ヶ月以上前のアーカイブをレビューしてください。

要件:
1. `99_Archive/` 内の全ファイルをスキャン
2. 3ヶ月以上前のファイルを抽出
3. 各ファイルの簡単な内容を表示
4. 削除候補を提案

手順:
1. `99_Archive/` フォルダ内をスキャン
2. ファイルパスから日付を抽出
3. 3ヶ月以上前かどうかを判定
4. ファイル内容を読み込んで簡単に要約
5. 削除候補を提案
6. ユーザーの承認を待つ

判断基準:
- 🗑️ 削除: 明らかに不要、薄い内容、重複
- ✅ 保留: まだ使える可能性がある
- 📝 Memoryへ: 見直しで価値が増した（04_Memory/へ移動）

## 使用例
```
/archive-review
```

## 期待される結果
- 3ヶ月以上前のアーカイブファイルがリストアップされる
- 各ファイルの内容要約が表示される
- 削除候補が提案される
- アクション（削除/保留/Memory移動）が表示される

## 注意事項
- 四半期ごと（3ヶ月に1回）に実行
- 積極的に削除することが重要
- Archiveは一時保管所であり、最終保存先ではない
