# 週次レビュープロンプト

## 使用タイミング
毎週日曜夜（30分）に週次レビューを実行する時

## プロンプト本文
週次レビューを実行してください。

実行内容:
1. Inbox整理レポート生成
2. 孤立ノート検出
3. 今週作成したノート数をカウント
4. タグ使用状況の分析
5. 03_Inputのクリーンアップ（1週間触っていないファイル）
6. 来週のアクション提案
7. 週次レビューファイル作成: `02_Daily/Weekly-Reviews/YYYY/YYYY-W[週番号].md`

手順:
1. 今週の日付範囲を計算（週番号も計算）
2. 作成されたノートをカウント
3. Inbox整理レポートを生成
4. 孤立ノートを検出
5. タグ使用状況を分析
6. 03_Input内の1週間触っていないファイルを検出
7. 週次レビューファイルを作成: `02_Daily/Weekly-Reviews/YYYY/YYYY-W[週番号].md`
8. 来週のアクションを提案

## 使用例
```
/weekly-review
```

## 期待される結果
- 週次レビューファイルが作成される
- 今週の実績がまとめられる
- Inbox整理レポートが生成される
- 孤立ノートが検出される
- タグ使用状況が分析される
- Inputクリーンアップが実行される
- 来週のアクションが提案される

## 注意事項
- 週番号の計算を正確に
- Inputクリーンアップは1週間触っていないファイルを対象に
