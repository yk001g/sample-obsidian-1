# AIツール比較リファレンス

| ツール | 強み | 弱み | 代表用途 |
|--------|------|------|----------|
| Cursor | IDE連携とAIペアプロ | 大規模PJで負荷増 | コーディング、レビュー |
| Claude | 長文・指示理解◎ | 画像入出力は限定的 | 仕様整理、ドラフト |
| Dify | ノーコードでエージェント | サーバ管理が必要 | PoC、社内Bot |
| Windsurf | ブラウザ完結 | エコシステムは少ない | 軽量コーディング |

## 選定観点
- セキュリティ: APIキー管理 / ログ暗号化
- UX: 学習コスト / 共同編集
- コスト: 席数・従量課金・上限制御
