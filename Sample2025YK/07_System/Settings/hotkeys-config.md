# ⌨️ ホットキー設定

**最終更新**: 2025-01-13

---

## 🎯 基本操作

### ファイル操作
- `Cmd/Ctrl + N`: 新規ノート作成
- `Cmd/Ctrl + O`: クイックオープン
- `Cmd/Ctrl + S`: 保存
- `Cmd/Ctrl + W`: タブを閉じる

### ナビゲーション
- `Cmd/Ctrl + P`: コマンドパレット
- `Cmd/Ctrl + E`: エディタ/プレビュー切り替え
- `Cmd/Ctrl + G`: グラフビュー
- `Cmd/Ctrl + Shift + F`: 全文検索

---

## 📝 編集操作

### テキスト編集
- `Cmd/Ctrl + B`: 太字
- `Cmd/Ctrl + I`: 斜体
- `Cmd/Ctrl + K`: リンク挿入
- `Cmd/Ctrl + Shift + K`: コードブロック

### 見出し
- `Cmd/Ctrl + 1-6`: H1-H6見出し
- `Cmd/Ctrl + 0`: 通常テキスト

---

## 🔗 リンク操作

### リンク作成
- `[[`: 内部リンク開始
- `Cmd/Ctrl + Click`: リンクを開く
- `Cmd/Ctrl + Shift + E`: リンクエディタ

### バックリンク
- `Cmd/Ctrl + Shift + B`: バックリンクパネル表示

---

## 🎨 表示操作

### パネル切り替え
- `Cmd/Ctrl + Shift + E`: エディタ/プレビュー
- `Cmd/Ctrl + Shift + L`: サイドバー表示/非表示
- `Cmd/Ctrl + Shift + M`: 左サイドバー表示/非表示
- `Cmd/Ctrl + Shift + R`: 右サイドバー表示/非表示

---

## 🔌 プラグイン操作

### Dataview
- `Cmd/Ctrl + Shift + D`: Dataviewクエリ実行

### Templater
- `Cmd/Ctrl + Shift + T`: テンプレート挿入

### Calendar
- `Cmd/Ctrl + Shift + C`: カレンダー表示

---

## 🎯 カスタムホットキー

### Dailyノート
- `Cmd/Ctrl + D`: Dailyノートを開く（カスタム）

### ダッシュボード
- `Cmd/Ctrl + Shift + H`: HOMEダッシュボードを開く（カスタム）

---

## 📱 モバイル操作

### 基本操作
- **長押し**: コンテキストメニュー
- **スワイプ**: パネル切り替え
