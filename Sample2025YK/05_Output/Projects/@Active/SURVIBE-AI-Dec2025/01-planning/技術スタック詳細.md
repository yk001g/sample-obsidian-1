# 技術スタック詳細

## Frontend

### Next.js 14+
- **App Router**: 最新のルーティングシステムを使用
- **Server Components**: デフォルトでServer Componentsを使用し、パフォーマンスを最適化
- **API Routes**: バックエンドAPIをNext.js内で実装

**参考**: [[Next.js-App-Router]], [[Next.js-Server-Components]]

### TypeScript
- 型安全性の確保
- 開発体験の向上
- エラーの早期発見

### Material-UI (MUI)
- **@mui/material**: コアコンポーネント
- **@emotion/react**: CSS-in-JS
- **@emotion/styled**: スタイリング

**使用コンポーネント例**:
- `Box`, `Container`: レイアウト
- `Typography`: テキスト
- `Button`: ボタン
- `TextField`: フォーム入力
- `Card`: カード
- `Grid`: グリッドレイアウト

### React Hook Form
- フォーム状態管理
- パフォーマンス最適化
- バリデーション統合

### Zod
- スキーマバリデーション
- TypeScript型推論
- エラーメッセージ管理

## Backend

### Next.js API Routes
- **Route Handlers**: `app/api/`ディレクトリでAPIを実装
- Server Componentsから直接データベースアクセスも可能

**参考**: [[Next.js-API-Routes]]

### MongoDB
- NoSQLデータベース
- 柔軟なスキーマ
- スケーラビリティ

### Mongoose
- MongoDB ODM（Object Document Mapper）
- スキーマ定義
- バリデーション
- ミドルウェア

**参考**: [[MongoDB]], [[Mongoose]]

## Infrastructure

### Vercel
- Next.jsに最適化されたデプロイプラットフォーム
- 自動デプロイ（GitHub連携）
- 環境変数管理
- サーバーレス関数

### MongoDB Atlas
- クラウドMongoDBサービス
- 無料プランあり
- 自動バックアップ
- スケーリング

## セットアップコマンド

```bash
# プロジェクト作成
npx create-next-app@latest survibe-ai-dec2025 \
  --typescript \
  --tailwind \
  --eslint \
  --app \
  --src-dir \
  --import-alias "@/*"

# 依存関係インストール
cd survibe-ai-dec2025
npm install @mui/material @emotion/react @emotion/styled
npm install mongoose zod react-hook-form @hookform/resolvers

# 開発サーバー起動
npm run dev
```
